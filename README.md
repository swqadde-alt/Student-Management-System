# Student Management System

## Description

This project is a console-based **Student Management System** developed in C as part of my programming learning journey. It allows users to manage student records through a menu-driven interface by registering students, viewing student information, calculating weighted averages, and displaying class statistics.

While developing this project, I focused on improving my understanding of functions, arrays, modular programming, and input validation. The project demonstrates the practical use of fundamental C programming concepts while emphasizing clean, readable, and well-organized code. Compared to my earlier projects, it reflects noticeable progress in both coding practices and problem-solving abilities.

## Features

* Register new students
* Display all student records
* Calculate weighted averages (40% Midterm, 60% Final)
* Display class statistics
* Identify the highest and lowest scoring students
* Input validation
* Menu-driven interface

## Technologies

* C Programming Language
* Standard C Library
* Arrays
* Functions
* Loops
* Conditional Statements
